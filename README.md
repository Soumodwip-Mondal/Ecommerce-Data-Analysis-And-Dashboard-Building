## 📌 Project Overview

This project analyzes E-Commerce sales data to uncover actionable business insights and build an interactive dashboard for performance monitoring.

The main goal of this analysis is to:
- Identify sales trends
- Understand customer purchasing behavior
- Analyze profitability by category & region
- Track key business KPIs
- Provide data-driven recommendations

This is an end-to-end data analytics project including data cleaning, analysis, visualization, and storytelling.
